# Apple Privacy Nutrition Labels

This document maps exactly to the App Store Connect Privacy disclosures based on the current LifeOS architecture.

## 1. Data Used to Track You
**None.** LifeOS does not link user data with third-party data for advertising, nor do we use tracking brokers.

## 2. Data Linked to You
The following data is collected and linked to the user's identity (if an account is created):

### Contact Info
- **Email Address**: Used for App Functionality (Account Authentication).

### User Content
- **Other User Content**: Journal entries, tasks, habits, and goals. Used strictly for App Functionality (Cloud Synchronization).

## 3. Data Not Linked to You
The following data is collected but anonymized/not linked to a specific user identity:

### Diagnostics
- **Crash Data**: Collected via Sentry for Diagnostics. Used to fix app instability. 
- **Performance Data**: Collected via Sentry for Diagnostics. 

---
*Note: If the user chooses to use LifeOS entirely offline without registering an account, absolutely zero data is linked to them.*
