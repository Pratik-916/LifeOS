# Store Publication Checklist

This checklist tracks the final legal and compliance inputs required for the Apple App Store Connect and Google Play Console dashboards.

## ✅ Completed (Ready for Input)
- [x] Privacy Policy Document Drafted
- [x] Terms of Service Drafted
- [x] Permission Audit Completed (No rogue permissions)
- [x] Third-Party SDK Audit Completed
- [x] Open Source Attribution Compiled

## ⚠️ Requires Manual Store Console Input
These items cannot be automated via code. A human must log into the respective store dashboards to execute them:
- [ ] **Data Safety Form (Google Play)**: Manually check the boxes corresponding to `google-play-data-safety.md`.
- [ ] **Privacy Nutrition Labels (Apple)**: Manually check the boxes corresponding to `apple-privacy-labels.md`.
- [ ] **IARC Content Rating**: Answer the questionnaire confirming the app contains no violence, profanity, or gambling.
- [ ] **Developer Contact Info**: Input `support@lifeos.io` and the support website.

## 🚀 Future Enhancements (Pre-Launch)
- [ ] Host the Privacy Policy on a live public URL (`https://lifeos.io/privacy`).
- [ ] Host the Terms of Service on a live public URL (`https://lifeos.io/terms`).
- [ ] Create the in-app "Open Source Licenses" screen in the Settings menu referencing `open-source-notices.md`.
