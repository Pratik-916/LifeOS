# Brand QA & Validation Checklist

Before finalizing any graphic asset or marketing copy, the designer or content manager must pass the asset through this QA checklist to ensure strict brand compliance.

## 1. Logo & Icon QA
- [ ] Does the App Icon pass the 80% safe area test?
- [ ] Is the clear-space padding respected around the wordmark?
- [ ] Are all in-app icons exactly 2px stroke width?
- [ ] Are active icons utilizing the Filled variant?

## 2. Store Assets QA
- [ ] Do the store screenshots follow the exact 5-shot storyboard?
- [ ] Is the screenshot caption typography highly legible on smaller mobile displays?
- [ ] Is the Feature Graphic completely devoid of text near the absolute edges (avoiding Android crop zones)?

## 3. Empty States & Voice QA
- [ ] Do all empty state illustrations use the approved flat-vector style?
- [ ] Does the empty state copy blame the user, or does it gently encourage them? (Ensure the latter).
- [ ] Is the marketing copy free of childish/gamified terminology?

## 4. Accessibility & Localization QA
- [ ] Do the selected colors pass WCAG AA contrast minimums?
- [ ] If localized, does the German/French text fit within the screenshot frames without breaking into a 3rd line?
- [ ] Are file naming conventions strictly adhered to (`ios-icon-1024x1024.png`)?
