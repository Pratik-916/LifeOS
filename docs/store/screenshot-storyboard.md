# Screenshot Storyboard

This storyboard defines a cohesive product narrative for the app store listings. The progression should guide the user from the macro view (Dashboard) down to the micro view (Journal/Analytics).

## Frame 1: The Unified View (Hero)
- **Screen**: Dashboard (Today View)
- **Mode**: Light Mode
- **Device**: iPhone 15 Pro Max (iOS) / Pixel 8 Pro (Android)
- **Caption**: "Master your day, your way."
- **Feature Highlighted**: The seamless integration of tasks, habits, and mood tracking on a single screen.

## Frame 2: Precision Planning
- **Screen**: Planner View
- **Mode**: Light Mode
- **Device**: iPhone 15 Pro Max / Pixel 8 Pro
- **Caption**: "Organize tasks with precision."
- **Feature Highlighted**: Time blocking, check-boxes, and due date badging.

## Frame 3: Lasting Routines
- **Screen**: Habit Tracker
- **Mode**: Dark Mode (Showcasing aesthetic flexibility)
- **Device**: iPhone 15 Pro Max / Pixel 8 Pro
- **Caption**: "Build habits that last."
- **Feature Highlighted**: Habit streaks, visual heatmaps, and completion circles.

## Frame 4: Mental Wellness
- **Screen**: Journal Editor
- **Mode**: Dark Mode
- **Device**: iPhone 15 Pro Max / Pixel 8 Pro
- **Caption**: "Reflect in a private space."
- **Feature Highlighted**: Distraction-free typography and clean editor UI.

## Frame 5: Long-Term Growth
- **Screen**: Analytics / Insights
- **Mode**: Light Mode
- **Device**: iPhone 15 Pro Max / Pixel 8 Pro
- **Caption**: "Track your progress over time."
- **Feature Highlighted**: Data visualization, completion charts, and offline-first privacy.
